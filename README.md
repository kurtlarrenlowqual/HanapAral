# 👥 Team Members & Roles (HanapAral)

### HanapAral – A Cloud Based Integrated Study Group Finder

Modern students often need digital tools that help them collaborate, organize study groups, and receive
timely notifications. Design and implement a mobile application for managing study groups that integrate
cloud-based services to support authentication, user profiles, group collaboration, and remote
configuration.
